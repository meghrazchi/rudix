Developer Quick Start

Ready to ship? Run the core engine locally and start indexing your data in minutes. Follow our surgical 4-step setup.

1

Clone the repository

Access the source on GitLab to get the latest stable build.

2

Configure Environment

Set your `.env` variables for database and API authentication.

3

Docker Compose

Spin up the required services including Postgres and Redis.

4

Run Migrations

Prepare your schema and start the developer server.
